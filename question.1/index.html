Explaination

In this test build 2 types of input,
- 1st one is get input word from html form and excute php script from html.

- 2nd one is running php script file from terminal, put arguments as input like

php q1.php arg1 arg2 arg3

How it works

This program is build for count string of last word. First step input accept the data we type. And split the data input into pieces using explode method, then we take last word with end method. After that passed data as a aurgument to length function.

Inside length function, start a for loop and iterates the string we passed before within strlen function,which returns the length of a string. And condition check every alphabet lettter between a~z or A~Z on each looping time. If a letter not a alphabet letter, the condition wont count the letter of string and export the length or count of alphabet string.

Remark

If you enters not letter of words like number or other character this porgram won't count. And also won't count the words combine with other characters such as "e!ght" "f1ve", will count the length of the alphabet letters only after special characters or numbers.
